# QORA — Overview

## Qué es

QORA (Quintana Operational Response Architecture) es una plataforma B2B SaaS para crear y operar agentes de voz con IA orientados a ventas, seguimiento comercial, calificación de leads y automatización de conversaciones.

Frase síntesis: "QORA es una plataforma para crear agentes de voz con IA que llaman, conversan, califican leads y registran resultados dentro de un sistema multi-cliente."

## Qué problema resuelve

| Problema | Cómo lo resuelve QORA |
|----------|----------------------|
| Alto costo de operadores | Agentes de IA escalables sin costo fijo por persona |
| Variabilidad en atención | Comportamiento consistente definido por configuración |
| Leads sin seguimiento | Scheduler automático con lógica de reintentos |
| Sin memoria entre llamadas | Memoria persistente por lead entre sesiones |
| Falta de trazabilidad | Transcripciones, análisis post-call y métricas |
| Contexto mezclado entre empresas | Arquitectura multi-tenant aislada por cliente |

## Qué hace diferente a QORA

No es "solo un agente de ElevenLabs". ElevenLabs da la capa de voz. QORA agrega la capa operativa: multi-tenant routing, CRM, memoria entre llamadas, scheduler, transcripciones, análisis post-call, herramientas conectadas al lead, prompts versionables por filesystem, dashboard y admin.

## Casos de uso

- Seguimiento y calificación de leads comerciales
- Ventas (piloto: seguros con Quintana Seguros / Jaumpablo)
- Agendamiento de reuniones o entrevistas
- Recontacto automático y recuperación de leads fríos
- Atención inicial y demos conversacionales de producto
- Agentes internos para responder preguntas sobre procesos

Seguros es el caso piloto principal, no el único uso posible.
